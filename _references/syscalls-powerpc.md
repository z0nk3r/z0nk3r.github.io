*AKA: `PowerPC`, `PPC`, `Power ISA`, `POWER`, `powerpc`.*

*CPU ISAs: `ppc`.*

## Calling Convention

| Syscall NR | Return | arg0 | arg1 | arg2 | arg3 | arg4 | arg5 |
|---|---|---|---|---|---|---|---|
| `r0` | `r3` | `r3` | `r4` | `r5` | `r6` | `r7` | `r8` |
{: style="--table-col-1: 12.5%; --table-col-2: 12.5%; --table-col-3: 12.5%; --table-col-4: 12.5%; --table-col-5: 12.5%; --table-col-6: 12.5%; --table-col-7: 12.5%; --table-col-8: 12.5%;"}

Invoked via `sc`. Errors are signaled via the SO (summary overflow) bit of the CR0 condition register, not a separate return register — `r3` holds the return value on success or a positive errno on failure. Note that `r3` doubles as both arg0 and the return register, since arg0 is overwritten by the time the syscall returns.
